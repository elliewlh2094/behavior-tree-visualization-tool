### T5 — File rename (click-to-edit)

**Description:** Make the file name text in the toolbar clickable to enter inline edit mode. Single click switches to an `<input>`, Enter/blur confirms, Escape cancels.

- [ ] Clicking the file name text switches to an `<input>` element
